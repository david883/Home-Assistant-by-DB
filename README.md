## 🚀 Projekt-Beschreibungen

### [Attendance Control](https://github.com/david883/Home-Assistant-by-DB/tree/main/blueprints%2Fautomation%2FDB%2FAttendance%20control)
Dieser Blueprint hilft dir dabei, die **Anwesenheit in deinem Zuhause** intelligent zu verwalten. Er bündelt verschiedene Sensoren, um zuverlässig festzustellen, ob Bewohner anwesend sind, und steuert darauf basierend z. B. Heizung, Licht oder Sicherheitsmodi.

### [Universal Status Monitoring](https://github.com/david883/Home-Assistant-by-DB/tree/main/blueprints%2Fautomation%2FDB%2FUniversal%20status%20monitoring)
Ein vielseitiges Tool zur **Zustandsüberwachung**. Hiermit kannst du beliebige Entitäten überwachen und wirst sofort benachrichtigt, wenn ein definierter Zustand (z. B. "Offline", "Batterie schwach" oder "Fenster offen") eintritt.

### 🛠️ Installation
1. Navigiere zum gewünschten Blueprint-Ordner.
2. Kopiere den Link zur `.yaml` Datei.
3. Gehe in Home Assistant auf **Einstellungen** > **Automatisierungen & Szenen** > **Blueprints**.
4. Klicke auf **Blueprint importieren** und füge die URL ein.
