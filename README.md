### 🚀 Project Descriptions

* **Attendance Control:** Aggregates various sensors to reliably determine if residents are home (useful for climate or lighting control).
* **Universal Status Monitoring:** Monitors any entity and notifies you of states like "Offline", "Low Battery", or "Window Open".
* **Zigbee Status Monitoring:** Specifically designed for Zigbee meshes to monitor connection quality (LQI) and report dropped devices.

### 🛠️ Installation
1. Copy the URL of the desired `.yaml` file (or the folder link).
2. In Home Assistant, go to **Settings** > **Automations & Scenes** > **Blueprints**.
3. Click **Import Blueprint** and paste the URL.
