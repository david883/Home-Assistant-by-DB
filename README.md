## 🚀 Beschreibungen

### [Attendance Control](https://github.com/david883/Home-Assistant-by-DB/tree/main/blueprints%2Fautomation%2FDB%2FAttendance%20control)
Präzise Verwaltung des Anwesenheitsstatus, um Automatisierungen wie Heizung, Licht oder Alarmanlagen zuverlässig zu steuern.

### [Universal Status Monitoring](https://github.com/david883/Home-Assistant-by-DB/tree/main/blueprints%2Fautomation%2FDB%2FUniversal%20status%20monitoring)
Ein flexibler Blueprint zur Überwachung beliebiger Sensoren oder Geräte. Erhalte Benachrichtigungen, sobald ein Gerät offline geht oder einen kritischen Wert erreicht.
