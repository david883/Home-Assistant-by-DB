## 🚀 Projekt-Beschreibungen

### Attendance Control
Dieser Blueprint ermöglicht eine präzise **Anwesenheitsverwaltung**. Er hilft dabei, Logiken zu vereinfachen, die den Status "Zuhause" oder "Abwesend" definieren, um daraufhin Sicherheitsmodi, Beleuchtung oder Heizprofile zu steuern.

### Universal Status Monitoring
Ein vielseitiges Werkzeug zur **Zustandsüberwachung**. Überwache beliebige Entitäten (z. B. Batteriestände, Online-Status oder Sensorwerte) und löse bei definierten Zustandsänderungen sofort Benachrichtigungen oder Folgeaktionen aus.

### 🛠️ Installation
1. Navigiere zum gewünschten Blueprint-Ordner.
2. Kopiere die URL der `.yaml` Datei.
3. Gehe in deinem Home Assistant auf **Einstellungen** > **Automatisierungen & Szenen** > **Blueprints**.
4. Klicke auf **Blueprint importieren** und füge die URL ein.
